# 🛒 Blinkit Sales Analysis Dashboard

## 📊 Project Overview

This project presents an interactive dashboard built using **Power BI** to analyze Blinkit grocery sales data.
The dashboard provides meaningful insights into sales performance, product categories, and outlet efficiency to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance
* Understand product category trends
* Compare outlet performance across tiers
* Identify customer purchasing patterns

---

## 🔍 Key Insights

* 📌 Total Sales: $1.20M+
* 📌 High contribution from **Low Fat products**
* 📌 Tier 3 outlets generate highest sales
* 📌 Top-performing categories: Fruits, Snacks, Household
* 📌 Variation in sales across item types and outlet sizes

---

## 📈 Features

* Interactive filters and slicers
* Dynamic visuals for better analysis
* Sales breakdown by:

  * Item Type
  * Fat Content
  * Outlet Tier
* Easy-to-understand layout for quick insights
